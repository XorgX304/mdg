About


Mock data generator is a free to use test data mocking web application. Generate up to 250,000 rows of random data at a time.

Technologies: Mock data generator uses Python, AWK, Flask, MongoDB, jQuery, JavaScript, Nginx and Gunicorn. Deployed on Google Compute Engine with file storing in Google Cloud Storage.

If you found a bug, an issue, have a question, suggestion, or just want to look at the source code, you can find everything at the Mock data generator GitHub repository.
